# My React App with Docker & Nginx

A production-ready React application built with **Vite**, containerized using **Docker**, and served through **Nginx**.

This project demonstrates how to package a React application into a Docker image and serve the optimized build using Nginx instead of the Vite development server.

---

## Features

* React + Vite application
* Multi-stage Docker build
* Lightweight production image
* Nginx web server
* Optimized static build
* Easy deployment using Docker

# How the Project Works

## Step 1: React Development

The application is developed inside the `src` folder.

Example:

```text
src/
    App.tsx
    main.tsx
    components/
```

During development you can run:

```bash
npm install
npm run dev
```

This starts the Vite development server.

---

## Step 2: Build the Application

Create an optimized production build:

```bash
npm run build
```

Vite generates the production files inside:

```text
dist/
```

The `dist` folder contains only static files such as:

* HTML
* CSS
* JavaScript
* Images
* Fonts

These files are ready to be served by any web server.

---

## Step 3: Docker Build

Build the Docker image:

```bash
docker build -t my-react-app .
```

The Dockerfile performs two stages:

### Stage 1

* Downloads Node.js dependencies
* Builds the React application
* Creates the `dist` folder

### Stage 2

* Uses a lightweight Nginx image
* Copies the build output into Nginx
* Starts the web server

This keeps the final image small because Node.js is not included in the production image.

---

## Step 4: Run the Container

Start the container:

```bash
docker run -d -p 8080:80 --name my-react my-react-app
```

Open your browser:

```
http://localhost:8080
```

---

# Nginx Configuration

The project includes a custom `nginx.conf`.

Its responsibilities include:

* Listening on port 80
* Serving files from the React build directory
* Returning `index.html` for client-side routes (when configured for SPA routing)
* Serving static assets efficiently

Example:

```nginx
server {
    listen 80;

    root /usr/share/nginx/html;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }
}
```

# Docker Commands

Build image:

```bash
docker build -t my-react-app .
```

List images:

```bash
docker images
```

Run container:

```bash
docker run -d -p 8080:80 --name my-react my-react-app
```

List running containers:

```bash
docker ps
```

Stop container:

```bash
docker stop my-react
```

Start container:

```bash
docker start my-react
```

Remove container:

```bash
docker rm -f my-react
```

Remove image:

```bash
docker rmi my-react-app
```
